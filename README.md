# 1XD3 Group 31

McMaster Computer Science course COMPSCI 1XD3 lab project.

## Structure

- `site/`: static HTML project pages and shared site assets.
- `concerthelper/`: PHP app for the McMaster Concert Band site.
- `database/`: importable MySQL schema and seed data.

## Entry Points

- Root landing page: `index.html` (redirects to `site/index.html`)
- Static site: `site/index.html`
- PHP app: `concerthelper/index.php`
- Database import: `database/concerthelper_schema.sql`
